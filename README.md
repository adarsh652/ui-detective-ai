# InspectUI 🔍

**Inspect. Understand. Build.**

InspectUI is a Chrome extension that helps frontend developers inspect any website, understand how modern interfaces are built, and recreate components faster.

## Features
- ⚡ **Tech Stack & Framework Detection**: Detects Next.js, React, Vue, Tailwind CSS, Framer Motion, and more on host pages.
- 🎯 **Non-blocking Inspector**: Hover over elements on any website to extract computed metrics (font size, weight, family, padding, margins, colors, border-radius).
- 🎨 **CSS to Tailwind Utility Converter**: Auto-converts element styles to standard Tailwind CSS classes (including `p-4`, `text-sm`, `font-semibold`, `rounded-lg`, custom hex colors `text-[#...]`, `bg-[#...]`).
- 📋 **One-Click Snippet Copy**: Copy generated Tailwind class strings and JSX code directly to your clipboard.
- 🛠️ **Component Recreation & Prompts**: Synthesizes 0ms client-side system prompts for v0, Cursor, or ChatGPT, and generates clean modular React JSX snippets.

## Installation
1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top-right corner.
4. Click **Load unpacked** and select the extension root directory.
5. Click the extension icon in the Chrome toolbar to open the Side Panel.
